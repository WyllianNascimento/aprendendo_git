O Git é uma excelente ferramenta para criar repositórios, sendo compatível com os principais sistemas operacionais, como: Windows, Linux e MacOs. Alguns de seus comandos são:

git init - cria um novo repositório do Git

git status - mostra as condições do diretório de trabalho]

git add + (nome do arquivo) - adiciona uma alteração arquivo especificado

git add . - adiciona uma alteração no diretório
